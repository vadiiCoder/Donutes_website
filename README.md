# 🍩 Donutes Website

A delicious and creative website filled with beautiful donut images and sweet design inspiration.  
This project was made for donut lovers and beginner bakers who enjoy cute and modern websites.

---

## ✨ About This Website

- 🍩 Showcases delicious donut images
- 🎨 Cute and modern UI design
- 💡 Gives inspiration for baking your own donuts
- 📱 Responsive and simple layout
- ⚡ Smooth and lightweight website experience

---

## 🛠️ Built With

- 👻 HTML5
- 🎀 CSS3
- ✨ JavaScript

---

## 🚀 How To Run

1. Download the ZIP file
2. Extract the project folder
3. Open `index.html` in your browser

Enjoy the website on your computer 🍩💻
